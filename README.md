# GitHub Automation & Alpaca Trading

A collection of scripts and documentation for GitHub automation and Alpaca trading.

## Features
- **GitHub Automation**: Use the GitHub CLI (`gh`) and API to automate tasks.
- **Alpaca Trading**: Scripts for trading automation and best practices.
- **Coding Best Practices**: Tools for linting, formatting, and debugging.

## Setup

### Prerequisites
- Python 3.10+
- Node.js (for JavaScript tools)
- Docker (optional)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/MalavyaRaval/github-automation-alpaca-project.git
   cd github-automation-alpaca-project
   ```

2. Install Python dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Install JavaScript dependencies:
   ```bash
   npm install --global eslint prettier
   ```

4. Set up environment variables:
   ```bash
   cp config/.env.example .env
   # Edit .env with your API keys
   ```
