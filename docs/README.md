# Project Documentation

This directory contains documentation for the **GitHub Automation & Alpaca Trading Project**.

## Files
- [GitHub_Automation.md](GitHub_Automation.md): GitHub CLI, API, and automation steps.
- [Alpaca_Trading.md](Alpaca_Trading.md): Alpaca trading best practices and scripts.
- [Coding_Best_Practices.md](Coding_Best_Practices.md): Linters, formatters, debuggers, and Docker.

## Overview
This project documents:
1. **GitHub Automation**: Using the GitHub CLI (`gh`) and API for automation.
2. **Alpaca Trading**: Best practices for Alpaca trading, including scripts and API usage.
3. **Coding Best Practices**: Tools like `black`, `flake8`, `debugpy`, and Docker for maintainable code.
